Dialogue Editor

Dialogue is a tool I made to write dialogue trees for my games. Edit and preview dialogue trees visually with a custom property system and XML based imports and exports. Dialogue is a WPF app written in C#.

Have a look at the GitHub repository.

Command

Command is a text based puzzle game that takes place entirely in terminal environment. Connect to the rebel mainframe and take them (and maybe someone else) down!

Command was made using Godot 3, with C# and gdscript, and blender.
